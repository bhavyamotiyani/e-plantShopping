# e-plantShopping

## Paradise Nursery Shopping Application

Paradise Nursery is an elegant, modern React-based web application built to bring a seamless shopping experience for plant enthusiasts. Whether you are looking for air-purifying indoor plants, aromatic herbs, natural insect repellents, medicinal plants, or low-maintenance greenery, Paradise Nursery provides a beautifully organized catalog to satisfy all your planting desires.

### Project Features
- **Landing Page**: An attractive and serene landing page that welcomes users with the theme "Where Green Meets Serenity" and includes a brief "About Us" overview.
- **Product Catalog**: Explore a curated collection of plants categorized by their benefits and characteristics. Each plant has an description, image, price, and a button to add to the cart.
- **Redux-Powered Shopping Cart**: Easily manage your cart with real-time updates. Add new plants, increment/decrement quantities, delete items, and view total cart costs dynamically.
- **Responsive Navigation**: A clean, premium green navigation bar allows swift transitions between the home page, product catalog, and shopping cart.

### Tech Stack
- **Framework**: React (Vite-based setup)
- **State Management**: Redux Toolkit (`@reduxjs/toolkit` and `react-redux`)
- **Styling**: Modern, premium CSS styling with rich color palettes and micro-interactions.

### How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/ibm-developer-skills-network/e-plantShopping.git
   cd e-plantShopping
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build and preview the project:
   ```bash
   npm run preview
   ```
